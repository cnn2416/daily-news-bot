# 🚀 Server酱完整配置指南

## 什么是Server酱？

Server酱是一个免费的微信消息推送服务，可以将消息推送到您的微信。对于每日新闻机器人来说，这是最方便的推送方式。

## 📱 配置步骤

### 第1步：注册和登录

1. **访问官网**：https://sct.ftqq.com/
2. **登录方式**：
   - 点击右上角"登录"
   - 选择"微信扫码登录"
   - 使用微信扫描二维码

### 第2步：获取SendKey

登录后，获取SendKey的三种方法：

#### 方法1：首页查看（最方便）
1. 登录后直接进入首页
2. 在页面顶部看到您的 **SendKey**
3. 格式：`SCUxxxxxxxxxxxxx`

#### 方法2：Key管理页面
1. 点击顶部导航栏 **"Key&设备"**
2. 在 **"SendKey"** 区域查看您的Key
3. 可以点击 **"复制"** 按钮复制Key

#### 方法3：API文档页面
1. 点击顶部导航栏 **"文档"**
2. 查看 **API文档** 中的示例代码
3. SendKey会显示在示例中

### 第3步：绑定微信

**非常重要！** 不绑定微信无法接收推送：

1. 在 **"Key&设备"** 页面
2. 找到 **"微信推送"** 部分
3. 点击 **"绑定微信"** 按钮
4. 扫码关注 **"Server酱"** 公众号
5. 完成绑定

### 第4步：测试推送

#### 在线测试：
1. 在首页找到 **"发送消息"** 区域
2. 输入标题和内容
3. 点击 **"发送"** 按钮
4. 查看微信是否收到消息

#### 命令行测试：
```bash
# 使用curl测试
curl -X POST "https://sctapi.ftqq.com/YOUR_SENDKEY.send" \
  -d "title=测试标题" \
  -d "desp=测试内容"
```

#### Python测试：
```python
import requests

sckey = "SCUxxxxxxxxxxxxx"  # 替换为您的SendKey
title = "测试标题"
desp = "测试内容"

url = f"https://sctapi.ftqq.com/{sckey}.send"
response = requests.post(url, data={"title": title, "desp": desp})
print(response.json())
```

## 🔧 在新闻机器人中配置

### 1. 创建配置文件
```bash
# 从示例文件创建配置
cp config/config.example.yaml config/config.yaml
```

### 2. 编辑配置文件
编辑 `config/config.yaml`：
```yaml
push:
  serverchan:
    enabled: true
    sckey: "SCUxxxxxxxxxxxxx"  # 您的SendKey
    title: "📰 每日新闻简报"
```

### 3. 验证配置
```bash
# 测试配置是否正确
python daily_tech_news_bot.py --test
```

## 🔍 常见问题

### 问题1：收不到推送
**解决方案**：
1. 确认微信已绑定Server酱公众号
2. 检查SendKey是否正确
3. 查看Server酱官网的消息记录
4. 检查网络连接

### 问题2：SendKey失效
**解决方案**：
1. 重新登录Server酱官网
2. 生成新的SendKey
3. 更新配置文件中的Key
4. 重新绑定微信

### 问题3：推送延迟
**解决方案**：
1. Server酱通常秒级推送
2. 偶尔延迟几分钟是正常的
3. 检查网络状况
4. 避免频繁测试推送

### 问题4：消息被拦截
**解决方案**：
1. 检查消息内容是否包含敏感词
2. 简化消息标题
3. 分段发送长消息
